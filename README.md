## Estrutura

O curso continha 4 avaliações: (1) envio de perguntas semanais
referentes aos textos de leitura obrigatória da semana; (2) a realização
de um seminário; (3) a realização de uma mini-pesquisa em grupo; (4) a
entrega de um trabalho final descrevendo apenas a metodologia de uma
pesquisa a ser realizada (sem necessidade de executar tal pesquisa).
Este pacote reúne as 3 últimas avaliações, que foram escritas em arquivo
Quarto e necessitaram de código. Todos os códigos podem ser encontrados
na pasta `\inst`, conforme a Tabela 2.

| Tipo              | Localização           |
|:------------------|:----------------------|
| Seminário         | `inst/seminario`      |
| Trabalho em grupo | `inst/trabalho-grupo` |
| Trabalho final    | `inst/trabalho-final` |
